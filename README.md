# docker
Docker files for development environment for python, audio analysis, computer vision and neural networks


- base files
- nvidia-docker

## Quick Start
```
git clone https://github.com/cyrta/docker
cd docker/base
./rebuild.sh
./start.sh
```
or just 
```
make
```

Then build the specific docker (i.e. computervision)
```
cd computervision/
./rebuild.sh
./start.sh
```
